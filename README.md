# How to Run the Code

This repository contains a Jupyter notebook and Python files implementing multi-armed bandit algorithms.

## Requirements

Install the required Python packages:

```bash
pip install numpy matplotlib jupyter
```

## Running the Code

1. Clone the repository:

```bash
git clone https://github.com/yourusername/repository-name.git](https://github.com/kaarlboma/homework3.git
cd homework3
```

2. Start Jupyter Notebook:

```bash
jupyter notebook
```

3. Open the notebook:

```
Programming Portion.ipynb
```

4. Run all cells in the notebook.

The notebook will:
- Import the algorithm implementations from the Python files
- Run the Monte Carlo simulations
- Compute expected regrets and standard errors
- Generate the plots comparing the algorithms
